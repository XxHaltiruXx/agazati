HTML Multimédia

Képek, videók és hangok

A HTML5 beépített támogatást nyújt multimédiás tartalmak megjelenítéséhez böngészőben, pluginok nélkül.

  • <img>: Képek beillesztése
  • <video>: Videók lejátszása
  • <audio>: Hangfájlok lejátszása
  • <iframe>: Beágyazott tartalom (YouTube, térképek)
  • <svg>: Vektorgrafika
  • <canvas>: Rajzolási felület JavaScript-tel

Példa: Multimédia elemek

Képek, videók és hangok használata HTML-ben: 

<!DOCTYPE html>
<html lang="hu">
<head>
    <meta charset="UTF-8">
    <title>Multimédia példa</title>
    <style>
        img { max-width: 100%; height: auto; border: 2px solid #ddd; }
        video, audio { width: 100%; max-width: 600px; }
    </style>
</head>
<body>
    <h1>Multimédia tartalom</h1>
    
    <h2>Kép</h2>
    <img src="https://via.placeholder.com/600x400" alt="Placeholder kép">
    
    <h2>Videó</h2>
    <video controls>
        <source src="video.mp4" type="video/mp4">
        A böngésződ nem támogatja a videó lejátszást.
    </video>
    
    <h2>Hang</h2>
    <audio controls>
        <source src="audio.mp3" type="audio/mpeg">
        A böngésződ nem támogatja a hang lejátszást.
    </audio>
    
    <h2>YouTube videó beágyazása</h2>
    <iframe width="560" height="315" 
        src="https://www.youtube.com/embed/dQw4w9WgXcQ" 
        frameborder="0" allowfullscreen>
    </iframe>
</body>
</html>
Próbáld ki magad!
