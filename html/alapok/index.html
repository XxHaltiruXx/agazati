HTML Alapok

Mi az a HTML?

A HTML (HyperText Markup Language) egy jelölőnyelv, a weboldalak alapja, amely a tartalom struktúráját és jelentését határozza meg (fejezetek, bekezdések, képek, linkek, táblázatok).

Alapvető HTML struktúra

Minden HTML dokumentum egy alap struktúrával kezdődik. A <!DOCTYPE html> meghatározza, hogy HTML5-öt használunk. A <html> tag a dokumentum gyökere, a <head> tartalmazza a metaadatokat, míg a <body>-ban helyezkedik el a látható tartalom. 

<!DOCTYPE html>
<html lang="hu">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Az oldalam címe</title>
</head>
<body>
    <h1>Helló Világ!</h1>
    <p>Ez az első HTML oldalam.</p>
</body>
</html>
Próbáld ki magad!

Alapvető HTML elemek

  • <h1> - <h6>: Címsorok (h1 a legnagyobb, h6 a legkisebb)
  • <p>: Bekezdés (paragraph)
  • <a href="...">: Link más oldalra
  • <img src="...">: Kép beszúrása
  • <div>: Általános konténer elem
  • <span>: Szövegrészlet megjelölése
  • <ul> / <ol>: Listakészítés (ul = számozatlan, ol = számozott)
  • <li>: Lista elem

Gyakorlati példa

Nézzünk egy összetettebb példát, amely bemutatja a címeket, bekezdéseket, listákat és linkeket: 

<!DOCTYPE html>
<html lang="hu">
<head>
    <meta charset="UTF-8">
    <title>Bemutatkozó oldal</title>
</head>
<body>
    <h1>Üdvözöllek az oldalamon!</h1>
    
    <h2>Rólam</h2>
    <p>Szia! Én vagyok XY, és szeretek programozni.</p>
    
    <h2>Érdeklődési körök</h2>
    <ul>
        <li>Webfejlesztés</li>
        <li>Python programozás</li>
        <li>Hálózati technológiák</li>
    </ul>
    
    <h2>Elérhetőségek</h2>
    <p>Keress meg a <a href="mailto:pelda@email.com">pelda@email.com</a> címen!</p>
</body>
</html>
Próbáld ki magad!
