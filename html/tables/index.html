HTML Táblázatok

Táblázat elemek

A táblázatok segítségével strukturált adatokat jeleníthetünk meg sorok és oszlopok formájában.

  • <table>: A táblázat konténere
  • <thead>: Táblázat fejléc
  • <tbody>: Táblázat törzs
  • <tfoot>: Táblázat lábléc
  • <tr>: Táblázat sor (table row)
  • <th>: Fejléc cella (table header)
  • <td>: Adat cella (table data)

Példa táblázat

Egy egyszerű táblázat diákok eredményeivel: 

<!DOCTYPE html>
<html lang="hu">
<head>
    <meta charset="UTF-8">
    <title>Osztályzatok</title>
    <style>
        table { border-collapse: collapse; width: 100%; }
        th, td { border: 1px solid #ddd; padding: 8px; text-align: left; }
        th { background-color: #4c0bce; color: white; }
        tr:nth-child(even) { background-color: #f2f2f2; }
    </style>
</head>
<body>
    <h1>Diákok osztályzatai</h1>
    
    <table>
        <thead>
            <tr>
                <th>Név</th>
                <th>Matematika</th>
                <th>Informatika</th>
                <th>Átlag</th>
            </tr>
        </thead>
        <tbody>
            <tr>
                <td>Kiss Anna</td>
                <td>5</td>
                <td>4</td>
                <td>4.5</td>
            </tr>
            <tr>
                <td>Nagy Péter</td>
                <td>4</td>
                <td>5</td>
                <td>4.5</td>
            </tr>
            <tr>
                <td>Szabó Kata</td>
                <td>5</td>
                <td>5</td>
                <td>5.0</td>
            </tr>
        </tbody>
    </table>
</body>
</html>
Próbáld ki magad!
