HTML Struktúra

Szemantikus HTML

A szemantikus HTML elemek világosan kifejezik jelentésüket mind a böngészőnek, mind a fejlesztőnek. Ezek az elemek segítenek strukturálni a tartalmat és javítják az oldalak hozzáférhetőségét.

  • <header>: Fejléc szakasz, általában logó, navigáció
  • <nav>: Navigációs linkek
  • <main>: Fő tartalom
  • <article>: Önálló tartalom (pl. blog poszt)
  • <section>: Tematikus csoportosítás
  • <aside>: Oldalsáv, kiegészítő tartalom
  • <footer>: Lábléc

Példa: Jól strukturált oldal

Így néz ki egy modern, szemantikus HTML struktúra: 

<!DOCTYPE html>
<html lang="hu">
<head>
    <meta charset="UTF-8">
    <title>Strukturált oldal</title>
</head>
<body>
    <header>
        <h1>Weboldal címe</h1>
        <nav>
            <a href="#home">Főoldal</a>
            <a href="#about">Rólunk</a>
        </nav>
    </header>
    
    <main>
        <article>
            <h2>Cikk címe</h2>
            <p>Cikk tartalma...</p>
        </article>
        
        <aside>
            <h3>Kapcsolódó</h3>
            <p>Oldalsáv tartalma</p>
        </aside>
    </main>
    
    <footer>
        <p>© 2025 Weboldal</p>
    </footer>
</body>
</html>
Próbáld ki magad!
