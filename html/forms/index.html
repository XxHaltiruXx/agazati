HTML Űrlapok

Űrlap alapok

Az űrlapok lehetővé teszik a felhasználók számára, hogy adatokat küldjenek a szervernek. A <form> elem tartalmazza az űrlap mezőit.

  • <input>: Beviteli mező (text, email, password, number, stb.)
  • <textarea>: Többsoros szövegmező
  • <select>: Legördülő lista
  • <button>: Gomb (submit, reset, button)
  • <label>: Címke input mezőkhöz

Példa űrlap

Egy egyszerű kapcsolatfelvételi űrlap példa: 

<!DOCTYPE html>
<html lang="hu">
<head>
    <meta charset="UTF-8">
    <title>Kapcsolat</title>
</head>
<body>
    <h1>Kapcsolatfelvétel</h1>
    
    <form action="/submit" method="post">
        <label for="name">Név:</label>
        <input type="text" id="name" name="name" required>
        <br><br>
        
        <label for="email">Email:</label>
        <input type="email" id="email" name="email" required>
        <br><br>
        
        <label for="message">Üzenet:</label><br>
        <textarea id="message" name="message" rows="4" cols="50"></textarea>
        <br><br>
        
        <label for="priority">Prioritás:</label>
        <select id="priority" name="priority">
            <option value="low">Alacsony</option>
            <option value="medium" selected>Közepes</option>
            <option value="high">Magas</option>
        </select>
        <br><br>
        
        <button type="submit">Küldés</button>
        <button type="reset">Törlés</button>
    </form>
</body>
</html>
Próbáld ki magad!
