HTML & CSS

HTML Alapok

A HTML (HyperText Markup Language) az internetes oldalak szerkezetének alapja.

<!DOCTYPE html>
<html lang="hu">
<head>
    <meta charset="UTF-8">
    <title>Példa oldal</title>
</head>
<body>
    <h1>Címsor</h1>
    <p>Ez egy bekezdés.</p>
</body>
</html>

CSS Alapok

A CSS (Cascading Style Sheets) felelős a weboldalak megjelenéséért.

body {
    background-color: #f0f0f0;
    font-family: Arial, sans-serif;
}

Box Modell

HTML elemek négy rétegből állnak: tartalom, padding, border, margin.

.doboz {
    width: 200px;
    padding: 10px;
    border: 2px solid black;
    margin: 10px;
}

Pozicionálás

.relative {
    position: relative;
    left: 50px;
}

.absolute {
    position: absolute;
    top: 20px;
}

Flexbox

A flexbox segítségével dinamikus elrendezéseket hozhatunk létre.

.container {
    display: flex;
    justify-content: space-between;
}

CSS Grid

.grid-container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 10px;
}

Responsív dizájn

A weboldalak különböző eszközökhöz való igazítása.

@media screen and (max-width: 600px) {
    body {
        background-color: lightblue;
    }
}

CSS animációk

@keyframes move {
    from { transform: translateX(0); }
    to { transform: translateX(100px); }
}

.animate {
    animation: move 2s infinite;
}
