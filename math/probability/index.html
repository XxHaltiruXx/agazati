Valószínűségszámítás

Mi az a valószínűség?

A valószínűség azt méri, hogy egy esemény mennyire várható, hogy bekövetkezzen. Értéke 0 és 1 között van (vagy 0% és 100% között).

  • 0 (0%): lehetetlen esemény
  • 0.5 (50%): egyenlő esély
  • 1 (100%): biztos esemény

Valószínűség számítása

P(A) = kedvező esetek száma / összes lehetséges eset száma

Példa 1: Mekkora a valószínűsége, hogy egy dobókockával 6-ost dobunk?

  • Kedvező esetek: 1 (csak egy 6-os van)
  • Összes eset: 6 (a kockán 1-6)
  • P(6) = 1/6 ≈ 0.167 ≈ 16.7%

Példa 2: Mekkora a valószínűsége, hogy páros számot dobunk?

  • Kedvező esetek: 3 (2, 4, 6)
  • Összes eset: 6
  • P(páros) = 3/6 = 1/2 = 50%

Kártyák valószínűsége

Egy 52 lapos francia kártyában:

  • 4 szín (kör, káró, treff, pikk)
  • Minden színből 13 lap (ász, 2-10, bubi, dáma, király)

Példák:

  • P(ász) = 4/52 = 1/13 ≈ 7.7%
  • P(piros lap) = 26/52 = 1/2 = 50% (kör vagy káró)
  • P(király) = 4/52 = 1/13 ≈ 7.7%

Események összeadása (VAGY)

Ha két esemény nem történhet meg egyszerre (kizáró események):

P(A vagy B) = P(A) + P(B)

Példa: Mekkora a valószínűsége, hogy 1-est vagy 2-est dobunk?

  • P(1) = 1/6
  • P(2) = 1/6
  • P(1 vagy 2) = 1/6 + 1/6 = 2/6 = 1/3 ≈ 33.3%

Független események szorzása (ÉS)

Ha két esemény független egymástól:

P(A és B) = P(A) × P(B)

Példa: Mekkora a valószínűsége, hogy két dobással mindkétszer 6-ost dobunk?

  • P(első 6) = 1/6
  • P(második 6) = 1/6
  • P(mindkettő 6) = 1/6 × 1/6 = 1/36 ≈ 2.8%

Kombinatorika - Permutáció

Permutáció: n elem összes lehetséges sorrendje. Jele: n! (n faktoriális)

n! = n × (n-1) × (n-2) × ... × 2 × 1

Példák:

  • 3! = 3 × 2 × 1 = 6
  • 4! = 4 × 3 × 2 × 1 = 24
  • 5! = 5 × 4 × 3 × 2 × 1 = 120

Alkalmazás: 3 ember hányféleképpen állhat sorba? 3! = 6 féleképpen

Kombinatorika - Variáció

Variáció: n elemből k elem kiválasztása, ahol a sorrend számít.

V(n,k) = n! / (n-k)!

Példa: 5 emberből 3-at kiválasztunk elnöknek, alelnöknek és titkárnak.

  • V(5,3) = 5! / (5-3)! = 120 / 2 = 60 féleképpen

Kombinatorika - Kombináció

Kombináció: n elemből k elem kiválasztása, ahol a sorrend NEM számít.

C(n,k) = n! / (k! × (n-k)!)

Példa: 10 emberből 3-at választunk ki egy csapatba (sorrend nem számít).

  • C(10,3) = 10! / (3! × 7!) = 120 féleképpen

Valószínűség a programozásban

A programozásban gyakran használunk véletlen eseményeket:

// Kockadobás szimulálása
function rollDice() {
  return Math.floor(Math.random() * 6) + 1;
}

// Esemény valószínűsége (kockadobás 1000x)
let sixes = 0;
for (let i = 0; i < 1000; i++) {
  if (rollDice() === 6) sixes++;
}
console.log("6-os esélye:", sixes / 1000);

// Faktoriális számítása
function factorial(n) {
  if (n === 0 || n === 1) return 1;
  return n * factorial(n - 1);
}

console.log(factorial(5)); // 120

  • Adatelemzés: Statisztikai modellek, predikciók
  • Biztonság: Kriptográfia, véletlenszám-generálás
  • Mesterséges intelligencia: Gépi tanulás, valószínűségi modellek
  • Pénzügy: Kockázatelemzés, befektetési döntések
