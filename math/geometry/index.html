Geometria

Mi az a geometria?

A geometria a matematika azon ága, amely az alakzatokkal, méretekkel, formákkal és az űrbeli tulajdonságokkal foglalkozik. Minden körülöttünk geometriai formákból épül fel.

Alapvető síkidomok - Négyszögek

Négyzet:

  • Mind a 4 oldala egyenlő, mind a 4 szöge 90°
  • Kerület: K = 4a (ahol a az oldal hossza)
  • Terület: T = a²
  • Példa: Ha a = 5 cm, akkor K = 20 cm, T = 25 cm²

Téglalap:

  • Szemközti oldalai egyenlők, mind a 4 szöge 90°
  • Kerület: K = 2a + 2b (a és b az oldalak)
  • Terület: T = a × b
  • Példa: Ha a = 6 cm, b = 4 cm, akkor K = 20 cm, T = 24 cm²

Háromszögek

A háromszög három oldallal és három szöggel rendelkező alakzat.

  • Kerület: K = a + b + c (három oldal összege)
  • Terület: T = (a × m) / 2 (alap × magasság / 2)
  • Szögek összege: Mindig 180°

Háromszögek fajtái:

  • Egyenlő szárú: két oldala egyenlő
  • Egyenlő oldalú: mind a három oldala egyenlő (minden szög 60°)
  • Derékszögű: egyik szöge 90°

Kör

A kör egy olyan alakzat, amelynek minden pontja egyenlő távolságra van a középponttól (sugár).

  • r: sugár (középponttól a körvonali pontig)
  • d: átmérő (d = 2r, a kör átellenes pontjai közötti távolság)
  • Kerület: K = 2πr vagy K = πd (π ≈ 3.14159)
  • Terület: T = πr²
  • Példa: Ha r = 3 cm, akkor K ≈ 18.85 cm, T ≈ 28.27 cm²

Pithagorasz-tétel

Derékszögű háromszögeknél az átfogó (c) négyzete egyenlő a két befogó (a és b) négyzetének összegével.

a² + b² = c²

Példa:

  • Ha a = 3 és b = 4, akkor c² = 9 + 16 = 25, tehát c = 5
  • Egy 3-4-5 háromszög mindig derékszögű!

Térbeli alakzatok - Kocka

A kocka hat egyenlő négyzetből álló testalakzat.

  • Felszín: F = 6a² (hat négyzet területe)
  • Térfogat: V = a³
  • Példa: Ha a = 4 cm, akkor F = 96 cm², V = 64 cm³

Térbeli alakzatok - Téglatest

A téglatest három különböző méretű téglalapokból épül fel.

  • Felszín: F = 2(ab + ac + bc)
  • Térfogat: V = a × b × c
  • Példa: Ha a = 3, b = 4, c = 5, akkor V = 60 cm³

Gömb

A gömb egy térbeli alakzat, ahol minden pont egyenlő távolságra van a középponttól.

  • Felszín: F = 4πr²
  • Térfogat: V = (4/3)πr³
  • Példa: Ha r = 3 cm, akkor F ≈ 113.1 cm², V ≈ 113.1 cm³

Henger

A henger két kör alappal és egy görbült oldallal rendelkezik.

  • Alapkör területe: A = πr²
  • Palást területe: P = 2πrh (h = magasság)
  • Teljes felszín: F = 2πr² + 2πrh = 2πr(r + h)
  • Térfogat: V = πr²h

Geometria a programozásban

A geometria elengedhetetlen a programozásban, különösen:

  • Játékfejlesztés: Ütközésérzékelés, objektumok pozícionálása
  • Grafika: 2D/3D renderelés, transzformációk
  • Webtervezés: Elrendezés, reszponzív design
  • Robotika: Navigáció, útvonaltervezés
// Kör területének számítása
function circleArea(radius) {
  return Math.PI * radius * radius;
}

// Pithagorasz-tétel
function pythagoras(a, b) {
  return Math.sqrt(a*a + b*b);
}

console.log(circleArea(5));    // 78.54
console.log(pythagoras(3, 4)); // 5
