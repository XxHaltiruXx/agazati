Függvények

Mi az a függvény?

A függvény egy szabály, amely minden bemeneti értékhez (x) pontosan egy kimeneti értéket (y) rendel. Jelölése: f(x) = ... vagy y = ...

Példa: f(x) = 2x + 3

  • Ha x = 1, akkor f(1) = 2·1 + 3 = 5
  • Ha x = 5, akkor f(5) = 2·5 + 3 = 13

Lineáris függvények

A lineáris függvény egy egyenes vonalat ír le a koordináta-rendszerben. Általános alakja: y = mx + b

  • m: meredekség (slope) - megmutatja, mennyire meredek az egyenes
  • b: y-tengely metszés (intercept) - ahol az egyenes metszi az y-tengelyt

Példák:

  • y = 2x + 1 → meredekség: 2, metszéspont: 1
  • y = -x + 5 → meredekség: -1, metszéspont: 5
  • y = 3x → meredekség: 3, metszéspont: 0 (origón keresztül megy)

Kvadratikus függvények

A kvadratikus függvény egy parabolát ír le. Általános alakja: y = ax² + bx + c

  • a > 0: felfelé nyíló parabola
  • a < 0: lefelé nyíló parabola
  • c: y-tengely metszéspont

Példák:

  • y = x² → alapparabola (felfelé nyílik)
  • y = -x² + 4 → lefelé nyíló parabola, csúcspontja (0, 4)
  • y = 2x² - 4x + 1 → felfelé nyíló, eltolt parabola

Gyökök és zérushelyek

A függvény zérushelye az a pont, ahol f(x) = 0, vagyis ahol a görbe metszi az x-tengelyt.

Példa: f(x) = x² - 4

  • Keressük, hol f(x) = 0: x² - 4 = 0
  • x² = 4
  • x = 2 vagy x = -2 (két zérushely)

Exponenciális függvények

Exponenciális függvényben a változó a kitevőben szerepel. Alakja: y = a·bx

  • b > 1: növekvő függvény (exponenciális növekedés)
  • 0 < b < 1: csökkenő függvény (exponenciális csökkenés)

Példák:

  • y = 2x → Minden lépésben megduplázódik (1, 2, 4, 8, 16, ...)
  • y = (1/2)x → Minden lépésben felezödik (1, 0.5, 0.25, 0.125, ...)
  • y = 100·1.05x → 5%-os éves növekedés (pl. kamat)

Logaritmikus függvények

A logaritmikus függvény az exponenciális függvény inverze. Alakja: y = loga(x)

Példák:

  • log10(100) = 2, mert 10² = 100
  • log2(8) = 3, mert 2³ = 8
  • ln(x) = természetes logaritmus (alap: e ≈ 2.718)

Függvények a programozásban

A programozásban a függvények kódblokkok, amelyek bemenetet kapnak és kimenetet adnak:

  • JavaScript példa:
function linear(x) {
  return 2 * x + 3;
}

function quadratic(x) {
  return x * x - 4 * x + 3;
}

console.log(linear(5));      // 13
console.log(quadratic(2));   // -1

Függvények alkalmazásai

  • Fizika: Mozgás, sebesség, gyorsulás számítása
  • Gazdaság: Ár-keresleti függvények, profitszámítás
  • Biológia: Populációnövekedés modellezése
  • Informatika: Algoritmusok komplexitása, adatmodellezés
  • Mérnöki: Pályagörbék, erő-elmozdulás függvények
