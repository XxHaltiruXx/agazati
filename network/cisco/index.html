Cisco alapok

Cisco IOS áttekintése

A Cisco IOS (Internetwork Operating System) a Cisco routerek és switchek operációs rendszere. Parancssori interfészen (CLI) keresztül konfiguráljuk az eszközöket.

IOS módok hierarchiája:

Mód Prompt Belépés Funkció
User EXEC Router> Alapértelmezett Alapvető monitoring parancsok
Privileged EXEC Router# enable Részletes információk, debug
Global Configuration Router(config)# configure terminal Eszköz szintű beállítások
Interface Configuration Router(config-if)# interface fa0/0 Interfész specifikus beállítások
Line Configuration Router(config-line)# line vty 0 4 Telnet/SSH beállítások
Router Configuration Router(config-router)# router ospf 1 Routing protokoll beállítások 
Módok közötti navigálás:
Router> enable                    (User → Privileged)
Router# configure terminal        (Privileged → Global Config)
Router(config)# interface fa0/0   (Global → Interface)
Router(config-if)# exit           (Vissza egy szinttel)
Router(config)# end               (Vissza Privileged-be)
Router# disable                   (Vissza User-be)

Alapvető router parancsok

1. Hostname és banner beállítása:

Router> enable
Router# configure terminal
Router(config)# hostname R1
R1(config)# banner motd #
Unauthorized access is prohibited!
#
R1(config)# end

2. IP cím beállítása interfészen:

R1# configure terminal
R1(config)# interface gigabitEthernet 0/0
R1(config-if)# ip address 192.168.1.1 255.255.255.0
R1(config-if)# description LAN Connection
R1(config-if)# no shutdown
R1(config-if)# exit

! Ellenőrzés
R1# show ip interface brief

3. Jelszavak beállítása:

! Enable jelszó (titkosított)
R1(config)# enable secret cisco123

! Console jelszó
R1(config)# line console 0
R1(config-line)# password console123
R1(config-line)# login
R1(config-line)# exit

! VTY jelszó (Telnet/SSH)
R1(config)# line vty 0 4
R1(config-line)# password telnet123
R1(config-line)# login
R1(config-line)# exit

! Jelszavak titkosítása
R1(config)# service password-encryption

4. Konfiguráció mentése:

! Mentés running-config → startup-config
R1# copy running-config startup-config
! vagy röviden:
R1# write
R1# wr

! Konfiguráció megtekintése
R1# show running-config
R1# show startup-config

Alapvető switch parancsok

1. VLAN létrehozása és interfész hozzárendelése:

Switch# configure terminal
Switch(config)# vlan 10
Switch(config-vlan)# name IT_Department
Switch(config-vlan)# exit

Switch(config)# vlan 20
Switch(config-vlan)# name Sales_Department
Switch(config-vlan)# exit

! Interfész hozzárendelése VLAN-hoz
Switch(config)# interface fastEthernet 0/1
Switch(config-if)# switchport mode access
Switch(config-if)# switchport access vlan 10
Switch(config-if)# exit

! Több interfész egyszerre
Switch(config)# interface range fa0/2-10
Switch(config-if-range)# switchport mode access
Switch(config-if-range)# switchport access vlan 20
Switch(config-if-range)# exit

2. Trunk konfiguráció:

! Trunk port beállítása (switch-switch vagy switch-router)
Switch(config)# interface gigabitEthernet 0/1
Switch(config-if)# switchport mode trunk
Switch(config-if)# switchport trunk allowed vlan 10,20,30
Switch(config-if)# switchport trunk native vlan 99
Switch(config-if)# exit

! Ellenőrzés
Switch# show interfaces trunk
Switch# show vlan brief

3. Management IP beállítása (SVI):

Switch(config)# interface vlan 1
Switch(config-if)# ip address 192.168.1.10 255.255.255.0
Switch(config-if)# no shutdown
Switch(config-if)# exit

Switch(config)# ip default-gateway 192.168.1.1

! Ellenőrzés
Switch# show ip interface brief

DHCP konfiguráció routeren

! DHCP pool létrehozása
R1(config)# ip dhcp pool LAN_POOL
R1(dhcp-config)# network 192.168.1.0 255.255.255.0
R1(dhcp-config)# default-router 192.168.1.1
R1(dhcp-config)# dns-server 8.8.8.8 8.8.4.4
R1(dhcp-config)# domain-name example.com
R1(dhcp-config)# lease 7
R1(dhcp-config)# exit

! Kizárt címek (pl. szervereknek, gatewaynak)
R1(config)# ip dhcp excluded-address 192.168.1.1 192.168.1.10

! DHCP relay (ha a DHCP szerver másik hálózaton van)
R1(config)# interface g0/0
R1(config-if)# ip helper-address 10.1.1.100
R1(config-if)# exit

! Ellenőrzés
R1# show ip dhcp binding
R1# show ip dhcp pool
R1# show ip dhcp conflict

SSH konfiguráció

! 1. Hostname és domain name beállítása
R1(config)# hostname R1
R1(config)# ip domain-name company.local

! 2. RSA kulcs generálása
R1(config)# crypto key generate rsa
How many bits in the modulus [512]: 2048

! 3. Felhasználó létrehozása
R1(config)# username admin privilege 15 secret Admin123!

! 4. SSH verzió és timeout beállítása
R1(config)# ip ssh version 2
R1(config)# ip ssh time-out 60
R1(config)# ip ssh authentication-retries 3

! 5. VTY vonalak konfigurálása
R1(config)# line vty 0 4
R1(config-line)# transport input ssh
R1(config-line)# login local
R1(config-line)# exit

! 6. Konzol timeout (opcionális)
R1(config)# line console 0
R1(config-line)# exec-timeout 5 0
R1(config-line)# exit

! Ellenőrzés
R1# show ip ssh
R1# show ssh

! SSH kapcsolat másik eszközről:
PC> ssh -l admin 192.168.1.1

Statikus routing

! Statikus útvonal hozzáadása
R1(config)# ip route 192.168.2.0 255.255.255.0 10.0.0.2
! Formátum: ip route [cél_hálózat] [maszk] [next-hop_IP]

! Alapértelmezett útvonal (default route)
R1(config)# ip route 0.0.0.0 0.0.0.0 203.0.113.1
! vagy interfész megadással:
R1(config)# ip route 0.0.0.0 0.0.0.0 serial0/0/0

! Floating static route (backup route magasabb AD-vel)
R1(config)# ip route 192.168.2.0 255.255.255.0 10.0.0.3 10

! Statikus útvonal törlése
R1(config)# no ip route 192.168.2.0 255.255.255.0 10.0.0.2

! Ellenőrzés
R1# show ip route
R1# show ip route static

Router-on-a-Stick (Inter-VLAN Routing)

! Router konfiguráció (subinterface minden VLAN-hoz)
R1(config)# interface gigabitEthernet 0/0
R1(config-if)# no shutdown
R1(config-if)# exit

! VLAN 10 subinterface
R1(config)# interface gigabitEthernet 0/0.10
R1(config-subif)# encapsulation dot1Q 10
R1(config-subif)# ip address 192.168.10.1 255.255.255.0
R1(config-subif)# exit

! VLAN 20 subinterface
R1(config)# interface gigabitEthernet 0/0.20
R1(config-subif)# encapsulation dot1Q 20
R1(config-subif)# ip address 192.168.20.1 255.255.255.0
R1(config-subif)# exit

! Switch trunk konfiguráció (router felé)
Switch(config)# interface gigabitEthernet 0/1
Switch(config-if)# switchport mode trunk
Switch(config-if)# switchport trunk allowed vlan 10,20
Switch(config-if)# exit

! Ellenőrzés
R1# show ip interface brief
R1# show vlans
Switch# show interfaces trunk

NAT (Network Address Translation)

1. Static NAT (egy-az-egyben):

! Belső cím leképezése külső címre
R1(config)# ip nat inside source static 192.168.1.10 203.0.113.10

! Interfészek megjelölése
R1(config)# interface g0/0
R1(config-if)# ip nat inside
R1(config-if)# exit

R1(config)# interface g0/1
R1(config-if)# ip nat outside
R1(config-if)# exit

2. Dynamic NAT (pool használatával):

! NAT pool létrehozása
R1(config)# ip nat pool PUBLIC_POOL 203.0.113.10 203.0.113.20 netmask 255.255.255.0

! Access list a belső címekhez
R1(config)# access-list 1 permit 192.168.1.0 0.0.0.255

! NAT pool és ACL összekapcsolása
R1(config)# ip nat inside source list 1 pool PUBLIC_POOL

! Interfészek
R1(config)# interface g0/0
R1(config-if)# ip nat inside
R1(config-if)# exit
R1(config)# interface g0/1
R1(config-if)# ip nat outside
R1(config-if)# exit

3. PAT/NAT Overload (legelterjedtebb):

! Sok belső cím → egy publikus cím (portokkal megkülönböztetve)
R1(config)# access-list 1 permit 192.168.1.0 0.0.0.255
R1(config)# ip nat inside source list 1 interface g0/1 overload

! Interfészek
R1(config)# interface g0/0
R1(config-if)# ip nat inside
R1(config-if)# exit
R1(config)# interface g0/1
R1(config-if)# ip nat outside
R1(config-if)# exit

! Ellenőrzés
R1# show ip nat translations
R1# show ip nat statistics
R1# clear ip nat translation *

Access Control Lists (ACL)

1. Standard ACL (csak forrás IP alapján):

! Standard ACL (1-99, 1300-1999)
R1(config)# access-list 10 permit 192.168.1.0 0.0.0.255
R1(config)# access-list 10 deny any

! Alkalmazás interfészre
R1(config)# interface g0/1
R1(config-if)# ip access-group 10 out
R1(config-if)# exit

! Ellenőrzés
R1# show access-lists
R1# show ip interface g0/1

2. Extended ACL (forrás, cél, protokoll, port):

! Extended ACL (100-199, 2000-2699)
R1(config)# access-list 100 permit tcp 192.168.1.0 0.0.0.255 any eq 80
R1(config)# access-list 100 permit tcp 192.168.1.0 0.0.0.255 any eq 443
R1(config)# access-list 100 deny ip any any

! Interfészre alkalmazás
R1(config)# interface g0/0
R1(config-if)# ip access-group 100 in
R1(config-if)# exit

3. Named ACL (ajánlott, szerkeszthető):

! Named Standard ACL
R1(config)# ip access-list standard ALLOWED_HOSTS
R1(config-std-nacl)# permit host 192.168.1.10
R1(config-std-nacl)# permit host 192.168.1.20
R1(config-std-nacl)# deny any
R1(config-std-nacl)# exit

! Named Extended ACL
R1(config)# ip access-list extended WEB_FILTER
R1(config-ext-nacl)# 10 permit tcp any any eq 80
R1(config-ext-nacl)# 20 permit tcp any any eq 443
R1(config-ext-nacl)# 30 deny ip any any
R1(config-ext-nacl)# exit

! Sorszám szerinti szerkesztés
R1(config-ext-nacl)# no 20
R1(config-ext-nacl)# 25 permit tcp any any eq 8080

! Alkalmazás
R1(config)# interface g0/1
R1(config-if)# ip access-group WEB_FILTER in
R1(config-if)# exit

ACL tippek:

  • Standard ACL-t a célhoz közel helyezd el
  • Extended ACL-t a forráshoz közel helyezd el
  • Implicit deny van az ACL végén (nem kell külön írni, de átláthatóság miatt jó)
  • Named ACL használata ajánlott (szerkeszthető, átlátható)

Diagnosztikai és troubleshooting parancsok

R1# show interfaces

R1# show ip interface g0/0

! Routing információk
R1# show ip route
R1# show ip protocols
R1# show ip route summary

! VLAN információk (switch)
Switch# show vlan brief
Switch# show vlan id 10
Switch# show interfaces switchport

! MAC address table
Switch# show mac address-table
Switch# show mac address-table dynamic

! ARP tábla
R1# show arp
R1# show ip arp

! CDP (Cisco Discovery Protocol)
R1# show cdp neighbors
R1# show cdp neighbors detail
R1# show cdp entry *

! Konfiguráció
R1# show running-config
R1# show startup-config
R1# show running-config interface g0/0

! Verzió és hardver információ
R1# show version
R1# show flash
R1# show processes cpu
R1# show memory

! Ping és traceroute
R1# ping 8.8.8.8
R1# ping 192.168.1.1 repeat 100
R1# traceroute 8.8.8.8

! Debug (óvatosan használd!)
R1# debug ip routing
R1# debug ip packet
R1# undebug all  (vagy: no debug all)
